## 🎯 Desktop Overview

DoaShow is a **desktop environment simulator** that replicates the experience of a modern operating system within your web browser. It features:

- **33+ Integrated Applications** - From media players to business tools
- **Floating Taskbar** - Draggable and customizable
- **Window Management** - Drag, resize, and organize multiple windows
- **Global Search** - Find and launch apps instantly
- **Dark Theme** - Cinema-inspired aesthetic with golden accents
- **Real-time Clock** - Always visible on the taskbar
- **Persistent Storage** - Your data is saved locally

### System Requirements

- **Browser**: Chrome, Firefox, Safari, or Edge (latest version)
- **Screen Size**: Minimum 1024x768 (recommended 1440x900+)
- **JavaScript**: Enabled
- **Storage**: 10MB+ localStorage available

## 📍 Taskbar Guide

### Taskbar Features

#### Start Menu (⊞)
Click to open the app launcher with:
- **Search Bar** - Type to find apps
- **Categories** - Browse by type
- **Recently Used** - Quick access to recent apps
- **All Apps** - Complete app list

#### Search Bar (🔍)
- **Real-time Search** - Results appear as you type
- **App Preview** - Hover to see app details
- **Quick Launch** - Press Enter to open first result
- **Clear Button** (✕) - Reset search

#### Pinned Apps
Default pinned apps:
- **🖼️ Slideshow** - Photo presentations
- **📝 Notes** - Note taking
- **📊 Calculator** - Basic calculations

Click any pinned app icon to launch it instantly.

#### Clock (🕐)
Shows current time in 12-hour format. Click to open Calendar app.

### Taskbar Customization

#### Moving the Taskbar
1. Click and drag the taskbar anywhere on screen
2. Position is saved automatically
3. Taskbar stays on top of all windows

#### Pinning Apps
1. Open App Launcher (Start Menu)
2. Hover over an app
3. Click "Pin to Taskbar" button
4. App appears in pinned section

#### Unpinning Apps
1. Right-click a pinned app
2. Select "Unpin from Taskbar"
3. App is removed from taskbar

## 🪟 Window Management

### Opening Windows

**Method 1: Start Menu**
1. Click ⊞ on taskbar
2. Search or browse for app
3. Click app name to launch

**Method 2: Search**
1. Click 🔍 on taskbar
2. Type app name
3. Press Enter or click result

**Method 3: Pinned Apps**
1. Click app icon on taskbar
2. Window opens instantly

### Window Controls

#### Moving Windows
- Click and drag the title bar
- Window follows your cursor
- Release to place

#### Resizing Windows
- Drag window edges or corners
- Resize in any direction
- Minimum size enforced

#### Maximizing Windows
- Double-click title bar
- Window fills screen
- Double-click again to restore

#### Minimizing Windows
- Click minimize button (-)
- Window hides from view
- Click taskbar icon to restore

#### Closing Windows
- Click close button (✕)
- Window closes and data is saved
- Reopen anytime

### Window Layering

- **Click to Focus** - Bring window to front
- **Auto-stack** - Windows don't overlap perfectly
- **Always on Top** - Taskbar stays above all windows

## 🐛 Troubleshooting

### Common Issues

**Apps Won't Open**
- Refresh the page
